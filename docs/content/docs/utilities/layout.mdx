Layout utilities provide comprehensive control over element positioning using flexbox and CSS grid. They enable rapid construction of complex layouts without custom CSS.

## Flexbox Direction

| Class | CSS Property |
|-------|--------------|
| `cat-flex-row` | `display: flex; flex-direction: row` |
| `cat-flex-row-rev` | `display: flex; flex-direction: row-reverse` |
| `cat-flex-col` | `display: flex; flex-direction: column` |
| `cat-flex-col-rev` | `display: flex; flex-direction: column-reverse` |

## Flex Properties

| Class | CSS Property |
|-------|--------------|
| `cat-flex-1` | `flex: 1 1 0%` (grow and shrink equally) |
| `cat-flex-auto` | `flex: 1 1 auto` (grow based on content) |
| `cat-flex-init` | `flex: 0 1 auto` (don't grow, can shrink) |
| `cat-flex-none` | `flex: none` (don't grow or shrink) |

## Flex Wrap

| Class | CSS Property |
|-------|--------------|
| `cat-flex-wrap` | `flex-wrap: wrap` |
| `cat-flex-nowrap` | `flex-wrap: nowrap` |

## Gap

Gap utilities control spacing between flex/grid items. Values correspond to the spacing token scale.

| Class | Description |
|-------|-------------|
| `cat-gap-xs` | Extra small gap |
| `cat-gap-s` | Small gap |
| `cat-gap-m` | Medium gap |
| `cat-gap-l` | Large gap |
| `cat-gap-xl` | Extra large gap |
| `cat-gap-2xl` | 2x extra large gap |
| `cat-gap-3xl` | 3x extra large gap |

### Directional Gap

| Class | Description |
|-------|-------------|
| `cat-gap-x-{size}` | Horizontal (column) gap only |
| `cat-gap-y-{size}` | Vertical (row) gap only |

### Auto-fit Grid

Create responsive grids that automatically adjust columns:

```html
<div class="cat-grid cat-grid-auto cat-gap-m" style="--grid-min: 200px;">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

## Examples

### Horizontal Navigation

```html
<nav class="cat-flex-row cat-gap-m cat-items-center">
  <a href="#">Home</a>
  <a href="#">About</a>
  <a href="#">Contact</a>
</nav>
```

### Card Layout

```html
<div class="cat-flex-row cat-gap-l cat-flex-wrap">
  <div class="cat-flex-1">Card 1</div>
  <div class="cat-flex-1">Card 2</div>
  <div class="cat-flex-1">Card 3</div>
</div>
```

### Centered Content

```html
<div class="cat-flex-row cat-justify-center cat-items-center" style="height: 200px;">
  <span>Centered content</span>
</div>
```

### Grid Dashboard

```html
<div class="cat-grid cat-grid-4 cat-gap-m">
  <div class="cat-grid-col-2">Wide widget</div>
  <div>Widget</div>
  <div>Widget</div>
  <div class="cat-grid-col-4">Full-width widget</div>
</div>
```

### Responsive Sidebar Layout

```html
<div class="cat-flex-col cat-flex-row:l cat-gap-l">
  <aside class="cat-flex-none" style="width: 250px;">
    Sidebar
  </aside>
  <main class="cat-flex-1">
    Main content
  </main>
</div>
```
