A dropdown component to display a list of actions in a dropdown menu or to
show additional content on demand.

## Placement

Use the `placement` attribute to control where the dropdown appears.

<LivePreview
  files={{
    "index.html": {
      code: `<cat-dropdown placement="bottom-start">
  <cat-button slot="trigger">Bottom Start</cat-button>
  <div>Content</div>
</cat-dropdown>

<cat-dropdown placement="top-end">
  <cat-button slot="trigger">Top End</cat-button>
  <div>Content</div>
</cat-dropdown>`
    },
    "styles.css": {
