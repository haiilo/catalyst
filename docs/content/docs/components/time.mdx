## With Min/Max Times

Use `min` and `max` attributes to restrict the time range.

<LivePreview
  files={{
    "index.html": {
      code: `<cat-time 
  label="Business hours" 
  min="09:00" 
  max="17:00"
></cat-time>`
    },
    "styles.css": {
      code: `html, body {
  display: flex;
  align-items: center;
  justify-content: center;
  min-height: 100%;
  gap: 1rem;
}`
    }
  }}
/>

## Step Intervals

Use the `step` attribute to set the interval between selectable times.

<LivePreview
  files={{
    "index.html": {

    }
  }}
/>

## Accessibility

- Always provide a label for the time input
- The time picker is keyboard accessible
