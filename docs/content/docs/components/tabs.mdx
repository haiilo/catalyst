Tabs are used to display multiple panels to be contained within a single
window, using tabs as a navigational element.

## Tabs Only Mode

Use `tabs-only` mode when you want to manage the content yourself.

<LivePreview
  files={{
    "index.html": {

## Accessibility

- Tabs implement the WAI-ARIA tabs pattern
- Keyboard navigation with arrow keys
- Active tab is indicated via `aria-selected`
