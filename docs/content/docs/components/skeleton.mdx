Skeletons are used to show where content will eventually be drawn, providing
a visual placeholder during loading states.

## Examples

<LivePreview
  files={{
    "index.html": {
      code: `<cat-skeleton></cat-skeleton>
<cat-skeleton variant="circle"></cat-skeleton>
<cat-skeleton variant="text"></cat-skeleton>`
    },
    "styles.css": {
      code: `html, body {
  display: flex;
  align-items: center;
  justify-content: center;
  min-height: 100%;
  gap: 1rem;
}`
    }
  }}
/>
