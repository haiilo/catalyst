Button groups are designed to bring together button controls that are of a
similar nature. For example, text formatting controls.

## Examples

<LivePreview
  files={{
    "index.html": {
      code: `<cat-button-group>

  <cat-button>Center</cat-button>
  <cat-button>Right</cat-button>
</cat-button-group>`

## Accessibility

- Button groups should have a logical ordering
- Each button should have an accessible label
