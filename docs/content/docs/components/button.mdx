Buttons are used for interface actions. Primary style should be used only
once per view for main call-to-action.

## Accessibility

- Always provide a visible label or `a11y-label` for icon-only buttons
- Use `disabled` state appropriately to indicate unavailable actions
- The button automatically manages focus states
