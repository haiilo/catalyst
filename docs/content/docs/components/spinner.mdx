## Sizes

Use the `size` attribute to control the spinner size.

<LivePreview
  files={{
    "index.html": {

<cat-spinner size="l"></cat-spinner>
<cat-spinner size="xl"></cat-spinner>`

    }
  }}
/>

## Accessibility

- Spinners have appropriate ARIA attributes to indicate loading state
- Use in combination with loading messages for screen readers
